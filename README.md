## 주요 기능

- 실시간 통신: webRTC 기술을 사용하여 여러 사람들이 이용하는데 강점이 있는 SFU 방식의 OpenVidu로 실시간 통신을 구현
- 공유 캔버스: canvas API를 활용한 공유캔버스에 리액트의 상태값으로 유저별 접근 권한을 조정하여 그림을 그리는 게임 구현
